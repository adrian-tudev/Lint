# Lint
Toy language for educational purposes trust. 

See the [Lint Language Reference](docs/reference.md) for more details.

## Installation
```
git clone https://github.com/adrian-tudev/Lint.git
cd Lint
mkdir build
cd build && cmake .. && make
```
An executable should be created called `lint` and can be ran with `./lint` on Unix/macOS or `.\lint` on Windows.
